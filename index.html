Game Quest Hub

GameQuest Hub is a modern, interactive web application built with React and Vite that serves as a central dashboard for gamers. It aggregates information on video games, including titles, genres, and release dates, offering a dynamic and visually engaging interface. Users can explore game details, track favorites, and access curated lists. The app emphasizes performance, responsive design, and intuitive navigation, leveraging React features such as hooks and state management to deliver a seamless user experience. GameQuest Hub showcases both frontend development skills and practical app design principles.

Book Finder App

Book Finder App is a React-based application that allows users to search for books by title, or author using the Google Books API. It features a clean UI with dynamic search results, detailed book modals, and accessibility enhancements like screen reader support. Users can view book thumbnails, descriptions, and direct purchase links via Amazon affiliate integration. The app uses lazy loading and Suspense to optimize performance and loading times. Favorites can be saved locally, with instant visual feedback and animations. Responsive design ensures usability across desktop and mobile devices. Search suggestions help refine queries based on user input. Built with React, Vite, React Router, and custom hooks for modularity and maintainability.

Weather App

This Weather App allows users to search for real-time weather conditions in any city worldwide. It fetches weather data using the WeatherAPI and integrates a location autocomplete feature powered by OpenStreetMap's Nominatim API. The app displays temperature, wind speed, humidity, conditions icon, and the local date, with interactive toggles to switch between °C/°F and km/h/mph. It features a responsive, grid-based layout styled with CSS animations and icons from Font Awesome, all wrapped in a clean and visually appealing UI.
